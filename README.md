# ✍️ AI Writing Assistant Bot

A powerful Telegram bot that uses Google Gemini AI to help you create, improve, and publish high-quality content directly from Telegram.

## ✨ Features

- **📝 Create Post** - Generate content from a topic description
- **✨ Improve Text** - Enhance structure, readability, and engagement
- **🔧 Fix Errors** - Correct grammar, spelling, and punctuation
- **✂️ Shorten** - Make text more concise
- **📝 Expand** - Add more details and examples
- **📊 Analyze** - Get detailed improvement recommendations
- **🌐 Translate** - Translate to different languages
- **📢 Direct Publishing** - Publish posts directly to your channels

## 🚀 Commands

- `/start` - Show main menu
- `/help` - Show help message
- `/publish @channel` - Publish content to a channel
- `/cancel` - Cancel current operation

## 🛠️ Setup

### Prerequisites

1. Telegram Bot Token from @BotFather
2. Google Gemini API Key from Google AI Studio (Free tier available)
